Quick Start

Integrate the GST Accelerator API into your app in under 5 minutes.

Base URL & Authentication

Include your API key in every request header

# Base URL
https://gstaccelerator.in

# Required header on all /api/v1/ endpoints
X-API-Key: YOUR_API_KEY

Code Examples

Pick your language

# HSN lookup
curl -X GET "https://gstaccelerator.in/api/v1/hsn/1001" \
  -H "X-API-Key: YOUR_API_KEY"

# Description search
curl -X POST "https://gstaccelerator.in/api/v1/lookup" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"description":"wheat flour","top_k":3}'
